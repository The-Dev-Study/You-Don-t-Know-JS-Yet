## 1. 클로저

책의 비동기 ajax 예제

```js
function getSomeData(url) {
    ajax(url, function callback(data) {
        console.log(data);
    });
}
```

-   부정적 의견 (원정, 동현): 예제가 헷갈렸다. 처음 보는 사람은 이해하기 어려울 듯
-   긍정적 의견 (성욱): 오히려 예제가 괜찮았다

> `getSomeData`는 실행 후 곧바로 종료되더라도, 비동기 콜백 함수는 렉시컬 스코프 체인을 통해 url 변수에 계속 접근 가능함을 보여주는 예제이다.

### 질문 (성욱): `var`와 함수 스코프

카운터 예제에서 var로 선언하면 전역 변수인데,...

```js
function counter() {
    var count = 0; // 함수 스코프 변수
    return function increment() {
        return ++count; // 클로저를 통해 count 접근
    };
}
```

-   var는 **함수 스코프(Function Scope)**를 따름
-   전역에서 선언된 var만 전역 변수가 됨
-   함수 내부에서 선언된 var는 해당 함수 스코프에 속함
-   변수 호이스팅: 선언부만 끌어올려지고, 할당은 원래 위치에서 실행됨

## this 키워드

### 질문 (성욱): 리덕스에서 화살표 함수와 function 키워드의 차이는?

```js
// function 키워드 - 동적 this
const obj = {
    name: 'test',
    method: function () {
        console.log(this.name); // 호출 컨텍스트에 따라 변경
    },
};

// 화살표 함수 - 렉시컬 this
const obj2 = {
    name: 'test',
    method: () => {
        console.log(this.name); // 상위 스코프의 this 사용
    },
};
```

> 💡 면접 팁: 사용하는 기술 스택을 미리 공부해 가는 것이 좋다. 관련 질문이 자주 나온다고 함!

## 프로토타입

### 프로토타입 섀도잉 (Prototype Shadowing)

-   하위 객체에서 상위 프로토타입의 프로퍼티/메서드를 오버라이딩 하는 것
-   프로토타입 체인에서 가까운 객체의 프로퍼티가 우선됨

```js
var parent = {
    greet() {
        return 'Hello';
    },
};

var child = Object.create(parent);
child.greet = function () {
    return 'Hi';
}; // 섀도잉 발생

child.greet(); // "Hi" (오버라이딩된 메서드 호출)
parent.greet(); // "Hello" (원본 유지)
```

### Object.create() 메서드

-   동현: 처음 봤는데 좋았다.
