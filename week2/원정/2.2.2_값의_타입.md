# 2.2.2 값의 타입
typeof 연산자를 사용해 원시 타입과 객체 타입 값을 구분한다.

``` js
typeof 42; // number
typeof "abc"; // string
typeof true; // boolean
typeof undefined; // undefined
typeof null; // object <- 버그다
typeof {"a": 1}; // object
typeof [1, 2, 3]; // object
typeof function hello(){}; // function
```

> typeof null은 예상과 달리 null이 아닌 object를 반환하고, 함수는 function을 반환하지만 배열은 array가 아닌 object를 반환한다.

문자열을 숫자로 바꾸는 것처럼 값의 타입을 변경하는 것을 JS에서는 타입 강제 변환이라고 한다.

변수에 할당하거나 어딘가에 넘길 때, 원싯값과 객체의 작동방식은 차이가 있다.
