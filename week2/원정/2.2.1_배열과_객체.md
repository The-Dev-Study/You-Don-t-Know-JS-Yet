# 2.2.1 배열과 객체
JS에는 원시 타입 말고 객체 타입도 존재한다.

배열은 특수한 유형의 객체인데, 객체 내 정렬된 데이터에는 숫자 인덱스가 매겨진다.

``` js
var names = [ "카일", "보라", "지수", "현" ];

names.length;
// 4

names[0];
// 카일

name[1];
// 보라
```

배열에는 원시 타입, 객체 타입 상관없이 모든 타입의 값이 들어갈 수 있다. 배열 안에 배열이 들어가는 것도 당연히 가능하다. 함수는 값이므로 배열이나 객체의 값에 함수 역시 할당할 수 있다.

객체는 배열보다 좀 더 일반적인 데이터 타입으로, 정렬되지 않은 키-값 쌍을 모아놓은 컬렉션이다. 배열은 인덱스라는 숫자를 통해 요소에 접근하는 반면 객체는 요소 위치를 알려주는 문자열(키 혹은 프로퍼티)을 사용해 요소에 접근한다.

``` js
var me = {
    first: "카일",
    last: "심슨",
    age: 39,
    specialties: [ "JS", "탁구" ]
};

console.log(`제 이름은 ${ me.first }입니다.`);
```
객체에서 원하는 정보에 접근하는 방식은 예시처럼 온점을 사용하는 방법 말고도 대괄호를 사용하는 방법도 있다.
