# 자바스크립트 이름의 유래

자바스크립트란 이름은 자바와 관련 X, 마케팅 목적으로 고안된 이름

### 왜 하필 자바스크립트?

자바 개발자들에게 어필하기 위해 + `스크립트`라는 단어가 당시 가벼운 프로그램이란 뜻으로 유행했기 때문

이렇게 만들어진 경량 스크립트인 자바스크립트는 웹이라는 신기술과 함께 우리에게 온 것

다만, 자바와 자바스크립트가 별개의 언어긴하지만 모두 C(C++)언어와 유사 문법을 기대하는 개발자를 대상으로 만들어졌기 떄문에 표면상 비슷하긴 함

- 블록 시작과 끝을 중괄호로 사용({})
- 문이 끝날 때 세미콜론(;) 사용

자바와 자바스크립트는 프로그래밍적 관점보단 법률적 관점에서 더 강하게 결부됨
(썬 마이크로시스템즈가 넷스케이프를 인수하며 자바스크립트를 비롯한 몇 기술을 함께 인수함 -> 이후 오라클이 썬 마이크로시스템즈 인수)

이러한 이유로 자바스크립트 대신 `JS`라는 이름을 사용하자고 제안하는 이들도 있음

오라클이 소유한 상표가 아닌 TC39에서 지정하고 ECMA 표준 기구에 의해 공식화된 명칭은 `ECMAScript` + 여기에 2016부터 개정 연도를 붇이기 시작 따라서 집필 시점인 2019년 자바스크립트의 공식 명칭은 ECMAScript 2019(ES2019)

`자바와 자바스크립트의 관계는 햄과 햄스터의 관계와 같다`
