# 1.4 JS의 다양한 얼굴

## 패러다임

코드를 어떻게 구조화할지에 대한 접근 방식과 사고방식을 의미

1. 절차적

- 탑다운(top-down)
- 선형적(linear)으로 구조화
- 프로시저(procedure)라 불리는 코드 단위에 미리 정해진 일련의 연산 작성
- C언어

2. 객체지향

- 클래스 기준으로 코드 구조화
- 클래스에는 로직과 데이터가 정의
- C++, 자바

3. 함수형

- 함수 단위로 구조화(이때 함수는 부수효과가 없는 순수함수)
- 함수 자체가 값으로 취급됨
- 하스켈(Haskell)

### 다중 패러다임 언어

여러 패러다임을 사용해 코드를 구조화  
 한 프로그램 안에서 두 개 이상의 패러다임에 입각한 코드를 작성할 수 있음

JS 또한 다중 패러다임 언어임
