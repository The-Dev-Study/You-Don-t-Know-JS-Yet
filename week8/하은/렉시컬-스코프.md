# 렉시컬 스코프

-   스코프 블록은 서로 중첩이 가능하다.
-   하나의 스코프 버블은 부모 스코프 버블에 온전히 포함된다. 따라서 한 스코프가 두 개의 바깥 스코프에 동시에 포함될 수 없다.

-   스코프는 어디에 정의되었느냐에 따라 달라지며, 어디서 접근 가능한지는 스코프 결정에 영향을 주지 않는다.

## 2.2 JS 엔진 구성원 간의 대화

-   엔진: 컴파일레이션을 시작부터 끝까지 책임지고 JS로 만든 프로그램을 실행함
-   컴파일러: 엔진의 친구로, 파싱과 코드 생성 과정에서 일어나는 모든 잡일을 담당
-   스코프 매니저: 선언된 모든 변수와 식별자를 담은 탐색용 목록을 작성하고 유지 보수 / 변수와 식별자 접근 관련 규칙을 강제함
